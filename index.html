Movimiento Rectilíneo Uniforme (MRU)

d = v * t

v = d / t

t = d / v

Movimiento Rectilíneo Uniformemente Acelerado (MRUA)

Vf = Vo + a*t

a = (Vf - Vo) / t

x = Vo*t + (a*t²)/2

x = ((Vf + Vo)/2) * t

Vf² = Vo² + 2*a*x
